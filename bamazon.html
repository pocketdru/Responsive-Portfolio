About the project

How it works

  1. The app shows a list of the available products.
  2. And offers the customer to buy listed goods.
  3. User can pick up the product by ID. And then they may choose how many units they want to buy.
  4. In the end, the app shows user's order.

  1. If a user selects View Products for Sale, the app should list every available item: the item IDs, names, prices, and quantities.

  2. If a user selects View Low Inventory, then it should list all items with an inventory count lower than five.

  3. If a user selects Add to Inventory, your app should display a prompt that will let the user "add more" of any item currently in the store.

  4. If a user selects Add New Product, it should allow the user to add a completely new product to the store.
